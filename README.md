The OP-MDT Contains 2 parts.
First the Backend Server created with Bun and Hono
Second the Frontend with React and Vite

To install dependencies:

```bash
bun i && cd ./client && bun i cd ../  
```

To run the server:

```bash
bun dev 
```

To run the client
```bash
bun client
```
